# 请求描述语法

## 简介

请求描述语法是一组请求动作的抽象，是 `string` 类型

## 语法

### 数据类型

#### 定义

数据类型将会包裹在一对尖括号内，例如 `<action>:hi` 则表示着，action可以替换为任何合法的action.

假设有一个类型定义为 `type action = 'say' | 'send'`  那么

```
say:hi
send:hi
```

上述都是合法的

### | 运算符

| 运算符被称之为或运算符，例如`A|B`表示，要么是A要么是B，不可以同时出现

### && 运算符
